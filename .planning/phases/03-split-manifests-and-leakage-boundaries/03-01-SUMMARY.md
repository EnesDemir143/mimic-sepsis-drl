## Design Decisions

- **Fixed seed persisted in manifest:** Every manifest records `seed` and `source_episode_set` so results are fully reproducible.
- **Polars DataFrame API:** Consistent with the existing episode pipeline (episodes.py, episode_models.py).
- **CLI dry-run fallback:** When the episode Parquet file does not exist yet, a synthetic 100-patient dataset is used ÔÇö safe for CI and Phase 4 development before real data is processed.
- **Parquet + JSON persistence:** Three per-split Parquet manifests + one `split_summary.json` in `data/splits/`.
